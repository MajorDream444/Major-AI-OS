## Purpose

MAIM 04 creates the reusable YouTube visual system for Major AI Mindset.

This system covers:

- YouTube channel banner
- YouTube profile / channel mark
- YouTube watermark
- Long-form thumbnails
- Shorts covers
- Live replay thumbnails
- Podcast episode covers
- Playlist covers
- Title/thumbnail testing
- Episode visual archive

The goal is not one-off thumbnails.

The goal is a repeatable visual language that turns every YouTube upload into a recognizable MAIM asset.

## Visual Doctrine

YouTube assets must feel:

```txt
Premium
Human
Cinematic
Readable at thumbnail size
Movement-coded
Beginner-safe
Founder-led
```

Avoid:

```txt
Generic robot graphics
Cold SaaS dashboards
Overloaded thumbnails
Unreadable small text
Random neon AI clichés
Tech-bro fear language
```

The hero is never AI.

The hero is the human learning to use AI.

## Thumbnail Types

### 1. Manifesto Thumbnail

Used for founder-message videos.

Example:

```txt
THE FUTURE NEEDS A TRANSLATOR
```

Visual pattern:

- Major on right or left
- Large type opposite him
- MAIM logo
- Episode label
- Three-part subtitle line
- Warm gold background story texture

### 2. ABC Lesson Thumbnail

Used for A-Z learning videos.

Pattern:

```txt
A IS FOR AWARENESS
B IS FOR BELIEF
C IS FOR CONTEXT
```

Visual pattern:

- Giant letter
- One strong lesson phrase
- MAIM tri-bar
- Small action / concept icon
- Founder or abstract lesson visual

### 3. Live Replay Thumbnail

Used for full webinar / show replays.

Pattern:

```txt
MAJOR AI MINDSET LIVE
EPISODE 01
SEE IT. BELIEVE IT. BUILD IT.
```

Visual pattern:

- Broadcast label
- Episode number
- Founder image
- Large episode title
- ABC concepts / icons
- AI is the Intelligence. You are the Wisdom.

### 4. Shorts Cover

Used for vertical clips from NotebookLM, Substack, webinars, and reels.

Pattern:

- One sentence only
- Vertical 9:16 safe zone
- Large readable type
- MAIM mark
- Minimal visual clutter

### 5. Playlist Cover

Used to organize the YouTube channel.

Initial playlists:

```txt
Start Here
The ABCs of AI Mindset
Major AI Mindset Live
AI for Real Life
Build With AI
The Future of Work
```

## Output Standard

Every thumbnail must pass this test:

```txt
Can a person understand it in under 2 seconds on a phone?
```

If not, simplify.

## Production Rule

Ready to publish does not mean complete.

A YouTube asset is ready when it has:

- readable title
- clear emotional promise
- MAIM mark
- correct visual world
- export stored in Drive
- status tracked in Airtable

A YouTube asset is complete when it also has:

- description
- tags
- transcript
- shorts cuts
- social captions
- analytics notes
- vault archive
