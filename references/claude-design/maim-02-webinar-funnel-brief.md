## Purpose

MAIM 02 turns the Learning Engine into the first live community conversion experience.

The webinar is not just a slide deck.

It is the doorway where strangers become learners, learners become community members, and community members become builders.

The first webinar should make AI feel safe, human, useful, and beginner-accessible.

## Slide Rule

Each slide should have:

```txt
One idea
Minimal text
Strong visual hierarchy
Speaker-safe structure
Beginner-safe language
```

Avoid dense slides.

Avoid jargon.

Avoid generic robot imagery.

Avoid hype.

## Teaching Rule

Every lesson section should include:

```txt
One concept
One example
One 5-minute action
One memory hook
```

## A-C Teaching Spine

### A — Awareness

Big idea:

```txt
You can't automate what you haven't noticed.
```

Teaching focus:

Before AI can help, you must notice the repeated tasks, energy leaks, bottlenecks, and unfinished loops in your life or business.

5-minute action:

```txt
Write down three tasks you repeat every week.
```

### B — Belief

Big idea:

```txt
If you can text, you can use AI.
```

Teaching focus:

Belief comes before skill. Beginners do not need to become technical first. They need permission to start.

5-minute action:

```txt
Ask AI one real question about your life, business, or idea.
```

### C — Context

Big idea:

```txt
Context creates clarity.
```

Teaching focus:

AI does not read your mind. It responds to the context you give it. Better input creates better output.

5-minute action:

```txt
Ask the same question twice: once vague, once with full context.
```
