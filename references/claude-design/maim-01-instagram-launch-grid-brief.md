## Visual System

### Feed Size

Use 4:5 Instagram feed format:

```txt
1080 x 1350
```

### Story/Reel Derivatives

Create matching 9:16 versions:

```txt
1080 x 1920
```

### Grid Behavior

When viewed as a profile grid, the first 9 posts should feel like a single system:

- Top row = entry points
- Middle row = education + belief
- Bottom row = roadmap + utility + founder story

### Visual Hierarchy

Each post should pass the five-second test.

A viewer should understand the post from the headline alone.

Use:

- Big bold headline
- One clear visual metaphor
- Minimal body copy
- Strong CTA
- Crown/tri-bar mark as identity anchor

## Design Direction

Create a premium but approachable look:

- Black editorial background
- Gold highlight frame or seal
- Green/blue/orange accents used sparingly
- Founder-led, community-centered, not tech-cold
- Mix of clean text cards, roadmap visuals, and human-centered AI imagery

Avoid:

- Too much tiny text
- Overloaded flyer designs for feed posts
- Generic robot imagery
- Corporate tech startup sameness
- Excessive neon cyberpunk

## Caption System

Every caption should follow this shape:

1. Hook
2. Relatable problem
3. Simple insight
4. One small action
5. CTA

Tone:

- Warm
- Direct
- Beginner-safe
- No shame
- No overwhelm
- Major’s voice
