Status: Version 1.0 Complete
Source: Claude Design

## Purpose

MAIM 03 is the interactive curriculum engine for Major AI Mindset.

It turns the ABCs from a content series into a reusable learning system that can power Instagram, webinars, workbooks, course lessons, podcasts, newsletters, community prompts, songs, voice narration, and future app experiences.

The Learning Engine is a design reference and curriculum architecture layer first.

It is not production code yet.

## Template Exploration

Claude Design explored three module-template directions:

1. Editorial Split
2. Workbook Page
3. Bento System

Decision:

```txt
Bento System = master engine
Editorial Split = hero energy
Workbook Page = printable / downloadable lesson energy
```

The Bento System was selected because it gives MAIM the most reusable structure for interactive navigation, lesson cards, future app conversion, and multi-format repurposing.

## Reusable Module Anatomy

Each lesson module now follows a 9-slot anatomy:

```txt
Letter + Word
The Question
The Metaphor
The Lesson
Mantra
What To Avoid
Reflect / Discuss
5-Minute Action
Memory Hook
```

A green `Next Level` nudge appears below the core lesson for learners ready to go deeper without overwhelming beginners.
