## MAIM 05 Sections To Build

### 1. Vertical Cover System

Create reusable 1080 × 1920 layouts for:

- Statement cover
- Founder cover
- Brand cover
- Lesson cover
- Quote cover
- CTA cover

Carry over MAIM 04 safe-zone logic:

- Top clear zone: 18%
- Bottom clear zone: 22%
- Critical text in center-safe area

### 2. First Frame System

The first frame of every vertical video must function like a thumbnail.

Rules:

- One sentence only
- Oversized type
- Strong face or strong metaphor
- No paragraph text
- No vague title
- Promise must be visible before audio starts

### 3. 10-Second Generation Rule

MAIM has a production constraint:

- 2 video generations per day
- 10 seconds each

Turn constraint into system:

> 1 Letter = 2 Hero Shots = Unlimited Edits

For every ABC letter:

- Shot A: emotional / metaphorical opening
- Shot B: practical teaching moment

Then edit with:

- Voiceover
- Music
- Text overlays
- Logo
- CTA

### 4. ABC Lesson Video Template

Every letter should have:

- Letter
- Big idea
- Visual metaphor
- One sentence lesson
- Five-minute action
- CTA

Example:

A — Awareness

> You cannot automate what you have not noticed.

B — Belief

> If you can text, you can use AI.

C — Context

> AI does not read your mind. It reads your context.

### 5. Founder Message Template

Use for big-idea / founder-to-camera videos.

Examples:

- The Future Needs a Translator
- AI Is a Confidence Machine
- You Don’t Need More Tools
- Likkle by Likkle, We Build the Future

### 6. Music Video Template

Use the ABC Theme system as the first case study.

Current locked concept:

> One Small Spark

Current prompt:

See `prompts/video-generation/abc-theme-one-small-spark-generate-prompt.md`

### 7. Caption + Subtitle Rules

- Captions must not conflict with platform UI
- Keep center-lower safe, but above the bottom UI zone
- Use high-contrast white / gold text
- Avoid long subtitle blocks
- Break lines naturally
- Use one emphasis color only
