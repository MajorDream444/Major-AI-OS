## A-C Slide Content

### A — Awareness

Headline:

```txt
A is for Awareness
```

Core line:

```txt
You can't automate what you haven't noticed.
```

Action:

```txt
Write down three tasks you repeat every week.
```

### B — Belief

Headline:

```txt
B is for Belief
```

Core line:

```txt
If you can text, you can use AI.
```

Action:

```txt
Ask AI one real question about your life, business, or idea.
```

### C — Context

Headline:

```txt
C is for Context
```

Core line:

```txt
Context creates clarity.
```

Action:

```txt
Ask the same question twice: once vague, once with full context.
```
