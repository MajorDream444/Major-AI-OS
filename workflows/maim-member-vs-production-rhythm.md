# MAIM Member Rhythm vs Production Rhythm

Date: 2026-06-27
Status: Decision locked
Related volumes:

```txt
MAIM 07 — Course + Cohort Assets
MAIM 08 — Hanzo Powered-By Assets
```

## Why This Exists

The weekly cadence was beginning to blur two different layers:

```txt
1. Member rhythm — what MAIM Cohort members experience.
2. Production rhythm — what Major / Hanzo / operators do behind the scenes.
```

These rhythms are related, but they should not be treated as the same schedule.

## Rule

```txt
MAIM 07 documents the member-facing delivery rhythm.
MAIM 08 documents the internal production rhythm and infrastructure support.
```

## Member Rhythm

This is the experience rhythm for members inside the MAIM Cohort.

```txt
Sunday — Major AI Mindset Live
Monday — Replay Available + Key Takeaways
Tuesday — AI Prompt Lab
Wednesday — Office Hours
Thursday — Implementation Day
Friday — Community Wins
Saturday — Reflection + Preparation
```

## Member Flywheel

```txt
Live Experience
→ Replay
→ Prompt Lab
→ Office Hours
→ Implementation
→ Community Wins
→ Reflection
→ Next Live
```

The live event launches the week. Everything after the live is designed to help members apply, ask questions, implement, reflect, and prepare for the next live.

## Production Rhythm

This is the internal rhythm for Major, Hanzo, and operators.

```txt
Monday — Review previous live + plan content
Tuesday — Record / generate content + prompt development
Wednesday — Office Hours support + production check
Thursday — Newsletter + community engagement
Friday — Final webinar prep + slides + tech check
Saturday — Promotion + reminders + rehearsal
Sunday — Major AI Mindset Live
```

## Why This Works

The sports metaphor:

```txt
Sunday = game day
Monday = film study
Tuesday = practice
Wednesday = coaching
Thursday = implementation
Friday = preparation
Saturday = anticipation
Sunday = next game
```

## What To Do In Claude Design

