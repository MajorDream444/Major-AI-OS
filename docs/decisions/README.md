# MAIM Decision Records

Status: foundation manual
Owner: Major AI Mindset

Decision records explain why important platform choices were made.

Docs explain how the platform works. Decision records preserve the reasoning that future agents and contributors should not have to rediscover.

## Format

Use ADR files:

```txt
ADR-0001-short-title.md
```

Each decision should include:

- Status
- Context
- Decision
- Consequences

## Current Records

```txt
ADR-0001-engine-first-architecture.md
ADR-0002-knowledge-separated-from-docs.md
ADR-0003-e-is-experiment.md
ADR-0004-landing-page-philosophy.md
ADR-0005-documentation-is-product.md
ADR-0006-preserve-ideas-implement-deliberately.md
ADR-0007-agent-workspace-architecture.md
```

## Rule

When a choice changes how future agents should think, build, test, or route work, create a decision record.
