# ADR-0006: Preserve Ideas Immediately, Implement Ideas Deliberately

Status: accepted
Date: 2026-07-07

## Context

MAIM generates many valuable ideas: product concepts, legacy infrastructure, knowledge vaults, agent systems, media engines, and business models.

If every strong idea becomes immediate implementation, active sprints lose focus. If ideas are not captured, the platform loses innovation.

## Decision

MAIM follows this engineering principle:

```txt
Preserve ideas immediately. Implement ideas deliberately.
```

## Consequences

- New ideas should be captured in the right layer quickly.
- Implementation waits for architecture, documentation, specification, verification, and release gates.
- Research and future concepts should not leak into production systems.
- The active season and sprint decide what gets built now.
- This keeps MAIM focused without losing its creative edge.
