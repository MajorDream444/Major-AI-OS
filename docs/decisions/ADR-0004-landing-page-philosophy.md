# ADR-0004: Landing Page Is an Invitation

Status: accepted
Date: 2026-07-07

## Context

MAIM social assets are strong, high-energy, and attention-grabbing. When translated too literally into a web page, they can make the landing page feel like a poster or ad instead of a trusted learning environment.

The landing page has a different job than the flyer.

## Decision

The MAIM Command Room landing page is an invitation, not an advertisement.

The approved design direction is:

```txt
65-70% command-room / elegant universe feel
30-35% MAIM brand asset language
```

## Consequences

- The page should feel like entering a room.
- The visitor should feel calmer after arriving.
- The page should teach before selling.
- The human is the hero; AI is the amplifier.
- Avoid duplicate poster treatments, heavy flyer energy, and generic AI marketing language.
- Future UI work must read `DESIGN.md` and `docs/design-system/`.
