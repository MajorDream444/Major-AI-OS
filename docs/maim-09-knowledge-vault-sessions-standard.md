## What This Standard Defines

A Knowledge Vault Session is a recorded teaching, conversation, webinar, keynote, workshop, podcast, live build, or coaching session that is intentionally captured and processed into reusable intellectual property.

This standard reframes the weekly webinar.

The session is not only a live event.

The session is the source asset for the MAIM Knowledge OS.

## Core Shift

Old framing:

```txt
Host a webinar.
Record it.
Post the replay.
```

New framing:

```txt
Run a Knowledge Vault Session.
Capture the wisdom.

```
