# MAIM Design System

Status: foundation manual
Owner: Major AI Mindset

This folder is the durable design source for MAIM product, landing page, content, and agent work.

Read this folder before generating UI, page copy, visual components, or brand-facing prototypes.

## Required Reading Order

1. `DESIGN.md`
2. `docs/design-system/maim-design-philosophy.md`
3. `docs/design-system/landing-page-philosophy.md`

## Purpose

The design system prevents redesign drift.

It keeps MAIM from becoming:

- generic AI marketing
- social graphics stretched into pages
- overbuilt dashboards without soul
- beautiful pages that do not explain what to do next

## Current Design North Star

MAIM is not only a website.

It is the Command Room: the front door to knowledge sessions, the ABC library, prompt systems, agents, labs, and the future Legacy OS.
