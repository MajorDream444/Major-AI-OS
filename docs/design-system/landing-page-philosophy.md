# MAIM Landing Page Philosophy

Status: canonical for public pages
Owner: Major AI Mindset

## Core Rule

The landing page earns trust.

The flyer gets attention.

Do not confuse those jobs.

## The Page Should Feel Like

- entering a room
- joining a knowledge session
- stepping into a command view
- meeting a human teacher
- finding a clear next step

## The Page Should Not Feel Like

- a social media flyer
- a generic AI course page
- a poster wall
- a stack of boxes
- a hype funnel

## Required Flow

```txt
Open the room
-> Show the next session
-> Explain the signal
-> Reveal the roadmap
-> Show the live rhythm
-> Let the visitor choose a lane
-> Introduce Major
-> Reserve the seat
-> Answer questions
```

## Visual Ratio

Use the current design ratio from `DESIGN.md`:

```txt
65-70% command-room / elegant universe feel
30-35% MAIM brand asset language
```

Use MAIM colors, mark, founder presence, and voice. Do not let social graphics become the page layout.

## CTA Rule

The main call to action should feel sovereign and calm.

Gold is the default action color. Orange is a hover or micro-accent, not the primary visual force.

## Roadmap Rule

Use one roadmap treatment at a time.

Do not repeat the ABC roadmap as a large poster and a smaller card in the same viewport.

## Copy Rule

Copy should sound like Major:

- direct
- useful
- warm
- grounded
- practical
- human

Avoid generic AI phrases that could belong to any SaaS page.
