## ABC Reframe

### A — Awareness

Old framing:

```txt
What AI tool should I use?
```

MAIM framing:

```txt
What am I not seeing?
Where is the opportunity?
Where is time leaking?
Where is the system failing?
```

### B — Belief

Old framing:

```txt
Can AI do this?
```

MAIM framing:

```txt
Can I see myself building this?
Do I believe my problem is worth solving?
Do I believe I can move from asking to creating?
```

Canonical beginner line:

```txt
If you can text, you can use AI.
```

### C — Context

Old framing:

```txt
What prompt should I type?
```

MAIM framing:

```txt
What problem am I really solving?
Where am I in the system?
Who is involved?
What would success look like?
What information does the tool need to help me well?
```

## Foundational Story

### The Uganda / Orphanage Conversation

A person called asking for money to support children in an orphanage.

The MAIM response was not simply to donate or dismiss the need.

The response was to reframe the problem using the ABCs.

```txt
Awareness:
There is already a community that cares.

Belief:
They can build something instead of only asking.

Context:
Church communities already buy physical goods and faith-centered products.
```

The practical idea became:

```txt
Create a print-on-demand product such as prayer pillows or faith-centered merchandise.
Use AI to generate mockups, product descriptions, social copy, and launch steps.
Sell the product and direct proceeds to the children.
```

The key teaching:

```txt
Do not only ask for help.
Create value that makes helping easier.
```

## Supporting Examples

### Puerto Rico IKEA Context Shift

The assumption was that IKEA worked the same way as on the U.S. mainland.

The reality was different: the Puerto Rico location operated more like a showroom / catalog process, requiring delivery and ordering instead of immediate pickup.

Lesson:

```txt
Context changes the strategy.
The same brand, same goal, and same person can produce different outcomes when the environment changes.
```
