# MAIM Agent Routing Data

Status: Phase 2 source data

This folder maps visitor lanes from the MAIM Command Room landing page to the future agent ecosystem.

Primary file:

```txt
data/agents/maim-agent-routing.json
```

The landing page currently asks for `Current lane`:

```txt
Beginner
Creator
Entrepreneur
Coach
Parent
Professional
```

Each lane should eventually route to:

```txt
Agent
Custom emails
Custom prompts
Custom lesson path
Downloads
ManyChat tag
CRM segment
Next CTA
```

This lets MAIM avoid sending every registrant the same follow-up.
