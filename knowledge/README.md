# MAIM Knowledge Layer

Status: foundation layer
Owner: Major AI Mindset

This folder contains what MAIM teaches.

Docs explain how the platform works. Knowledge contains the lessons, stories, prompt systems, session material, and reusable teaching assets that the platform delivers.

## Rule

Knowledge assets should remain reusable.

If a lesson, prompt, story, or playbook can be taught again, route it here instead of burying it in a one-off document.

## Relationship to Docs

```txt
docs/       = how MAIM operates
knowledge/  = what MAIM teaches
data/       = structured machine-readable records
prompts/    = agent and generation instructions
workflows/  = repeatable production processes
```
