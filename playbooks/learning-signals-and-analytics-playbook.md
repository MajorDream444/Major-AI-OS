# Learning Signals & Analytics Playbook

## Purpose

This playbook defines how Major AI Mindset evaluates whether the ABC content is actually teaching, not just getting views.

The system does not optimize only for virality.

It optimizes for:

```txt
Attention
→ Application
→ Trust
→ Conversion
→ Community
```

## KPI Layer 1 — Hold Quality

Hold quality tells us whether the content earned attention.

Track:

- 3-second hold rate
- Average watch time
- Completion rate
- Rewatch rate

Interpretation:

| Signal | Meaning | Action |
|---|---|---|
| Low 3-sec hold | Hook is weak | Rewrite first line / visual open |
| Good hold, poor completion | Pacing drags | Tighten edit / shorten body |
| High completion | Format works | Reuse structure |
| High rewatches | Dense value or unclear point | Turn into carousel or longer breakdown |

## KPI Layer 2 — Depth of Engagement

Depth of engagement tells us whether the content was useful.

Track:

- Saves
- Shares
- Comments
- Profile visits
- Link clicks
- Keyword CTA comments

Interpretation:

| Signal | Meaning | Action |
|---|---|---|
| High saves | Useful reference | Turn into carousel/checklist |
| High shares | Socially valuable | Expand into community prompt |
| High comments | Emotional resonance | Build conversation post |
| High profile visits | Brand curiosity | Strengthen bio/landing path |
| High keyword comments | Conversion interest | Route to Opal app / email list |

## KPI Layer 3 — Learning Signals

Learning signals are qualitative evidence that the audience understood and applied the lesson.

These matter more than vanity metrics.

## What Counts As A Learning Signal

A learning signal includes:

1. A viewer paraphrases the lesson in their own words.
2. A viewer reports completing the five-minute action.
3. A viewer asks an implementation question.
4. A viewer shares a specific use case.
5. A viewer helps another commenter understand the idea.
6. A viewer asks for a deeper version.
7. A viewer says the concept reduced fear or confusion.

## What Does Not Count

These are engagement signals, not learning signals:

- Fire emojis
- “Great post”
- “This is dope”
- Generic praise
- Spam comments
- One-word reactions

## Learning Signal Examples

### Beginner Signal

```txt
I tried asking AI to rewrite my client email and it helped me make it clearer.
```

Meaning:

The user applied the five-minute action.

### Medium Signal

```txt
Could I turn this into a repeatable workflow for my team?
```

Meaning:

The user is ready for Medium tier content.

### Expert Signal

```txt
How would this work if we need governance and compliance for employees using AI?
```

Meaning:

The user is ready for Expert tier or consulting/enterprise pathway.

## Decision Rules

### If a letter gets high saves

Create:

- Carousel
- Checklist
- Newsletter
- Medium tier workflow

### If a letter gets high shares

Create:

- Community discussion prompt
- Reel follow-up
- Story post

### If a letter gets high learning signals

Create:

- Medium version
- Expert version
- Podcast episode
- Workshop segment

### If a letter gets low hold but strong comments

The idea is strong but the hook/edit failed.

Action:

- Rewrite hook
- Reshoot shorter version
- Keep core concept

### If a letter gets high views but no saves/comments

The content entertained but did not teach.

Action:

- Add clearer action step
- Add CTA
- Create practical version

## Week 6 Consolidation Metrics

At the end of Beginner A-Z, review:

- Highest 3-second hold
- Highest completion rate
- Most saved
- Most shared
- Most commented
- Most keyword comments
- Most learning signals
- Most common objections
- Best candidate for Medium expansion
- Best candidate for Expert expansion
- Best candidate for Opal app
- Best candidate for paid workshop

## Conversion Signals

Treat these as strong buyer/community intent:

- “How do I learn this?”
- “Do you teach this live?”
- “Can you help my business?”
- “Where do I start?”
- “Is there a course?”
- “Can you build this for me?”
- “Can my team learn this?”

Route these to:

```txt
DM
→ Opal mini-app
→ email list
→ 7-Day Lab
→ Leverage Cohort
→ consulting / Hanzo pathway
```

Examples:

- Potential partner
- Strong community member
- Possible client
- Teacher/mentor candidate
- Repeat commenter
- Person who helped others
