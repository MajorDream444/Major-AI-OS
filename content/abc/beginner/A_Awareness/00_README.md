# A = Awareness

A is the opening lesson in the Major AI Mindset ABC series.

Core doctrine:

Awareness comes before automation.

This package is the canonical V2 production folder for the A lesson. It is designed for Google Veo/Gemini/Omni generation, local asset storage, future Remotion assembly, and later use by Claude Code and Codex.

## Lesson Summary

Most people think AI starts with prompts. It does not. It starts with awareness.

Before learners automate anything, they must notice where time, energy, and attention are leaking.

## Production Rule

One letter equals two 10-second hero shots.

Shot A is the emotional metaphor.

Shot B is the practical teaching moment.

Together they become one 20-30 second social reel and a reusable cinematic asset for the MAIM Visual Engine.
