# SYSTEM CORE — MAJOR AI OS

## Primary Stack
- ChatGPT = Brain, strategy, writing, research partner
- Codex = Builder, executor, repo worker, PR machine
- OpenClaw = Agent runtime and orchestration layer
- Hanzo = Agent intelligence and ecosystem layer
- Lux = rails, trust, settlement, tokenization layer
- GitHub = Source of truth for code, docs, tasks, handoffs
- Airtable = Business data and CRM
- Notion = Human dashboard and operating docs
- Obsidian + Graphify = Memory graph and context compression
- n8n = Automation, triggers, webhooks, scheduled work
- NotebookLM = Research vault and source synthesis

## Operating Law
1. Always read SYSTEM_CORE.md first.
2. Always reference GitHub as the source of truth.
3. Never rely on chat memory alone.
4. Every task must map to a workflow.
5. Every output must be logged.
6. Every agent must leave a handoff.
7. Prefer reusable systems over one-off outputs.
8. Do not rebuild what already works; integrate and improve.

## Execution Loop
Plan → Build → Run → Log → Improve
