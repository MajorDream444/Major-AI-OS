# MAIM Founders + ABC Shirt Prompt Pack

Status: Active concept pack
Date: 2026-06-26
Tool target: Gemini / Nano Banana / image mockup tools

## Merch Doctrine

MAIM merch should not feel like generic AI apparel.

It should feel like movement artifacts.

People should wear the ideas because the ideas represent who they are becoming.

## Core Phrase Update

Approved phrase:

```txt
AI is the Intelligence. You are the Wisdom.
```

This belongs in the Founder Collection and should appear in future slides, shirts, posters, and closing visuals.

## Visual Rules

Use the MAIM brand system:

```txt
Black foundation
Gold authority
Jamaican green legacy
Knicks blue / orange champion energy
White clarity
Crown coin seal as primary mark
Green MD mark sparingly / OG use only
```

Avoid generic robot graphics.
Avoid cold tech-bro aesthetics.
Avoid clutter.
Avoid fake luxury.

Merch should feel:

```txt
Premium
Clean
Wearable
Culturally grounded
Human-centered
Movement-coded
```

## Mockup Notes

For Nano Banana / Gemini:

- Then generate back design if the front is strong.
- Keep text readable.
- Use one phrase per shirt.
- Avoid overdecorating.
- Prefer black shirts first.
- Add white/cream and green variants later.
