# MAIM Gravity Well Landing Page Experiment

Status: Experiment
Owner: Major Dream Williams
Branch: feature/maim-gravity-well-landing-test

## Purpose

This experiment tests whether the Gravity Well visual template should become the opening hero for the Major AI Mindset landing page.

The goal is not to replace the full MAIM website.

The goal is to test whether a cinematic 3D universe-style hero can work as the first doorway into the MAIM ecosystem while the rest of the page stays clear, fast, and conversion-focused.

## Strategic Read

Major AI Mindset is not just a webinar or content brand.

It is a mindset operating system for the AI age.

That means the landing page should feel like people are entering a new way of thinking, not just registering for another Zoom.

The Gravity Well pattern is promising because it visually communicates:

- Universe
- Orbit
- Gravity
- Attraction
- Thought systems
- Inner architecture
- A living learning path

This matches the MAIM philosophy: the mind is a universe, and the ABCs are the first map.

## Decision Rule

Use Gravity Well as the MAIM Universe Hero only.

Do not turn the entire landing page into a 3D experience.

The 3D layer should create wonder.

The page copy should create clarity.

The signup flow should create action.

## Test Versions

### Version A — Lightweight MAIM Landing Page

No Three.js.

Uses static gradient, glow, crown seal, and simple animated background.

Purpose:

- Fastest loading
- Best mobile safety
- Strong conversion baseline

### Version B — Gravity Well Hero

Uses Three.js only in the opening hero.

The rest of the page remains standard HTML sections.

Purpose:

- Test emotional impact
- Test premium universe effect
- Test whether the visual increases signups or distracts
