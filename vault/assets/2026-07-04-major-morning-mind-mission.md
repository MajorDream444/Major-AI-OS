## Core Thesis
AI does not replace thinking. It accelerates the loop:

```txt
Awareness → Context → Belief → Experiment → Reflection → New Awareness
```

The ABCs are not a straight line. They are a living thinking engine.

## Strategic Notes
The morning format should stay raw. The robe, tea, and hand-on-forehead frame are signals of honesty, not mistakes. It positions Major as a builder thinking out loud before the world gets noisy.

This episode should teach the difference between using AI to extract and using AI to explore.

Most users do this:

```txt
Idea → Prompt → Answer → Done
```

Builders do this:

```txt
Awareness → Conversation → Context → Questions → Challenge assumptions → Refine → Prototype → Test → Reflect → Asset
```
