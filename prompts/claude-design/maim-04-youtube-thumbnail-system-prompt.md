# Claude Design Prompt — MAIM 04 YouTube + Thumbnail System

Copy this into Claude Design.

---

Create a new project called:

```txt
MAIM 04 YouTube + Thumbnail System
```

Use MAIM 00 Brand System as the parent visual reference.

This project is the reusable YouTube identity system for Major AI Mindset.

It should support:

- YouTube channel banner
- Channel avatar / profile mark
- YouTube watermark
- Long-form video thumbnails
- Live show replay thumbnails
- ABC lesson thumbnails
- Shorts covers
- Playlist covers
- Title/thumbnail A-B test boards
- Export checklist

## Visual References To Use

Use the uploaded assets as the current design direction:

- YouTube channel banner
- The Future Needs a Translator thumbnail
- MAIM crown seal logo
- transparent MAIM watermark logo
- founder portrait / thumbnail composition
- MAIM 02 Live deck screenshots
- MAIM 03 Learning Engine screenshots

## Brand Rules

Use this visual doctrine:

```txt
Black foundation
Gold authority
Jamaican green legacy
Champion blue action energy
Orange call-to-action energy
White clarity
Crown coin seal as primary trust mark
```

Core identity:

```txt
Major AI Mindset
The Future Needs a Translator
See it. Believe it. Build it.
AI is the Intelligence. You are the Wisdom.
One Letter. One Lesson. One Action.
Likkle by likkle, we build the future.
```

## Design Feel

The YouTube system should feel like:

```txt
Premium broadcast package
Founder-led education channel
Human-centered AI movement
Caribbean diaspora + New York championship energy
Cinematic but clear
Readable on mobile
Confident, not hypey
```

Avoid:

```txt
Generic robots
Cold AI stock imagery
Overcrowded text
Tiny unreadable labels
Neon tech-bro cliches
Dystopian fear aesthetic
```

## Build These Pages / Boards

### 1. Channel Banner System

Create 2 banner variations:

1. Full cinematic founder banner
2. Cleaner text-forward banner

Required banner copy:

```txt
MAJOR AI MINDSET
AI is the Intelligence. You are the Wisdom.
See it. Believe it. Build it.
One Letter. One Lesson. One Action.
Likkle by likkle, we build the future.
@major_ai_mindset
majoraimindset.com
```

Make sure the safe-zone center works for YouTube desktop, mobile, and TV crop.

### 2. Manifesto Thumbnail Template

For founder-message videos.

Example title:

```txt
THE FUTURE NEEDS A TRANSLATOR
```

Create 3 variations:

- Founder right, headline left
- Founder left, headline right
- Big type only with crown seal

### 3. ABC Lesson Thumbnail Template

Create templates for A, B, C.

Use:

```txt
A IS FOR AWARENESS
B IS FOR BELIEF
C IS FOR CONTEXT
```

Each should include:

- giant letter
- one memorable phrase
- MAIM tri-bar
- small concept icon
- crown seal

### 4. Live Replay Thumbnail Template

For Major AI Mindset Live.

Include:

```txt
MAJOR AI MINDSET LIVE
EPISODE 01
SEE IT. BELIEVE IT. BUILD IT.
AI IS THE INTELLIGENCE. YOU ARE THE WISDOM.
```

### 5. Shorts Cover Template

Vertical 9:16 covers.

Rules:

- one sentence only
- large text
- clear MAIM mark
- safe upper/lower space
- can work as Instagram Reel cover and YouTube Shorts cover

### 6. Playlist Cover Templates

Create covers for:

```txt
Start Here
The ABCs of AI Mindset
Major AI Mindset Live
AI for Real Life
Build With AI
The Future of Work
```

### 7. A-B Test Board

Create a board that compares:

- Title variant
- Thumbnail variant
- Emotional hook
- Target audience
- Notes

### 8. Export Checklist

Create a final page that lists export requirements:

```txt
YouTube thumbnail: 1280 x 720
YouTube banner: 2560 x 1440
Shorts cover: 1080 x 1920
Profile/avatar: 800 x 800
Watermark: transparent PNG under 1MB
```

## Design Rule

Every thumbnail must pass this test:

```txt
Can a viewer understand the promise in under 2 seconds on a phone?
```

If not, simplify.

## Final Mood

This should feel like the official YouTube broadcast package for a movement that teaches AI with humanity, culture, clarity, and ownership.

Likkle by likkle, we build the future.
