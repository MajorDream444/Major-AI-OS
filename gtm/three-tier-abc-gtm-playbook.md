# MAIM GTM Playbook — Three-Tier AI Mindset ABCs

## Source

Built from the June 21, 2026 NotebookLM / audio deep dive on the Three-Tier AI Mindset ABC framework.

## Core Thesis

Major AI Mindset is not just a content series.

It is a modular educational operating system.

Treat every ABC lesson as:

```txt
Reusable curriculum
+ Social media asset
+ Podcast topic
+ Mini-app prompt
+ Video prompt
+ Community challenge
+ Lead magnet
+ Future cohort entry point
```

## Architectural Metaphor

Physical architecture is rigid. A house does not become a skyscraper because an expert walks in.

Digital education can be different.

The same foundation can feel like a simple starter home for a beginner and scale vertically into a skyscraper for experts.

That is the purpose of the Three-Tier ABC model.

## Audience Problem

The beginner audience is not blocked mainly by software.

They are blocked by psychology:

- Low technical confidence
- Fear of being replaced
- Decision paralysis
- Belief that AI is only for coders
- Overwhelm from too many tools
- Fear of participating in their own obsolescence

The GTM strategy must lead with emotional safety before technical complexity.

## Core Beginner Promise

```txt
You can do this.
```

The beginner tier moves people from:

```txt
Fearful / hesitant exploration
→ safe simple application
```

The first win is not mastery.

The first win is a five-minute action that proves AI responds to plain language.

## Three-Tier Structure

| Tier | Main Verb | Promise | Cognitive Role |
|---|---|---|---|
| Beginner | Notice | You can do this | Personal empowerment |
| Medium | Build | You can improve this | Workflow optimization |
| Expert | Design | You can systematize this | Strategic architecture |

## Bloom's Taxonomy Alignment

The system intentionally climbs the learning pyramid:

```txt
Beginner = remember / understand / apply
Medium   = apply / analyze / build
Expert   = evaluate / design / create
```

Do not push beginners into expert-level execution too early.

## Letter Spine Rule

The A-Z spine stays consistent across all tiers.

What changes:

- Depth
- Vocabulary
- Proof
- Action
- Visual badging
- Platform packaging

Example:

```txt
A Beginner = Notice 3 draining tasks
A Medium   = Build an awareness audit across workflows
A Expert   = Design AI opportunity mapping by function, margin, and risk
```

## Rollout Modes

### Mode 1 — Back-to-Back Mode

Publish all 26 beginner letters first.

Then repeat the alphabet at Medium depth.

Then repeat the alphabet at Expert depth.

Best for:

- Maximum beginner clarity
- Community onboarding
- Avoiding confusion

### Mode 2 — Beginner-Led Simultaneous Mode

Publish beginner letters daily.

Add medium/expert depth as weekly opt-in layers.

Recommended cadence:

```txt
Monday: Beginner Reel
Wednesday: Medium Carousel / Workflow Post
Friday: Expert LinkedIn Document / Deep Dive
```

Avoid dropping all three tiers on the same day.

## Recommended GTM Choice

Use **Beginner-Led Simultaneous Mode**.

Reason:

- Beginner reels drive mass reach.
- Medium content captures builders.
- Expert documents position Major as strategist/architect.
- Keeps the feed clear without wasting the deeper intellectual property.

## Six-Week Beginner Launch

### Week 0 — Setup

- Trailer
- Design system locked
- Templates built
- A-E scripts ready
- Airtable / Notion / GitHub prepared
- Cowork daily operator activated

### Week 1 — A to E

Launch foundational letters.

Goal:

- Build trust
- Reduce fear
- Establish the daily rhythm

### Week 2 — F to J

Goal:

- Maintain cadence
- Begin tracking saves/shares
- Watch comments for learning signals

### Week 3 — K to O

Goal:

- Add first LinkedIn recap document
- Begin mid-season synthesis
- Package early lessons into newsletter/podcast

### Week 4 — P to T

Goal:

- Emphasize practical workflows
- Push users toward first application

### Week 5 — U to Z

Goal:

- Finish the alphabet
- Increase CTA toward waitlist/community

### Week 6 — Consolidation Week

Do not launch new letters.

Use this week for:

- A-Z recap reel
- A-Z carousel summary
- Analytics review
- Learning signal review
- Medium tier teaser
- Expert waitlist call
- Community reflection

Week 6 is not a pause. It is the conversion and consolidation layer.

## Platform Strategy

### Master Asset

Always start with a vertical 9:16 video master.

Ideal length:

```txt
30–60 seconds
```

Reason:

- Instagram discovery favors short reels.
- TikTok rewards first-second clarity.
- YouTube Shorts can use slightly longer/searchable variants.
- LinkedIn can receive reframed professional/contextual versions.

### Repurposing Rule

Do not reshoot every platform.

Repackage the same core content DNA.

| Platform | Adaptation |
|---|---|
| Instagram Reels | Short, clear, visual hook |
| TikTok | Fast first-second clarity, direct action |
| YouTube Shorts | SEO title + pinned prompt/comment |
| LinkedIn | Professional framing + PDF carousel/deep insight |
| Newsletter | Reflection, story, and next step |
| Podcast | 10–20 minute conceptual expansion |

## Learning Signal Framework

Track three KPI layers:

1. Hold quality
2. Depth of engagement
3. Learning signals

### Hold Quality

Measures:

- 3-second hold
- Completion rate

Tells you:

- Hook quality
- Editing pace
- Visual clarity

### Depth of Engagement

Measures:

- Saves
- Shares
- Comments
- Profile clicks

Tells you:

- Practical usefulness
- Topic value
- Shareability

### Learning Signals

Measures qualitative evidence:

- Viewer paraphrases lesson in their own words
- Viewer reports completing the five-minute action
- Viewer asks an implementation question
- Viewer helps another commenter
- Viewer asks for next tier / deeper example

Learning signals determine what becomes Medium/Expert content.
